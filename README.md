## 📂 Understanding the Files

The folder you downloaded contains three main sub-folders. Each folder focuses on a different workshop task.

### Picking the Right Model
The `rightmodel` folder teaches you how to test different AI models. You learn how to check the quality of answers against the cost of the model. This helps you choose the best setup for your budget. You refine settings like effort levels and thinking parameters to get faster or better results.

### Agent Decomposition
The `agent-decomposition` folder shows you how to break down large tasks. Instead of one long instruction, you split the work into smaller actions. You learn to use skills and agents to manage specific parts of a project. This makes your code cleaner and easier to test at every step.

### How We Claude Code
The `how-we-claude-code` folder guides you through a full product workflow. You start with an idea, create design drafts as simple web pages, and finish with a working web application. This project uses modern tools like Vite and React to build components. It shows you how to bridge the gap between a design and a finished app.
